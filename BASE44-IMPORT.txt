Projects — 1:1 Code-Paket fuer Base44
=====================================

Inhalt: kompletter Quellcode der App (Frontend + Server + Electron), ohne node_modules.

WICHTIG — So uebertraegt Base44 Code 1:1:
Base44 hat KEINEN direkten ZIP-Upload fuer beliebige Custom-Apps.
Der offizielle Weg fuer Frontend-Code ist: GitHub verbinden.

Variante A (empfohlen — 1:1 in Base44):
1. Dieses ZIP entpacken.
2. Ordner "Projects" als GitHub-Repo hochladen (neues privates Repo).
3. In Base44: + → "Migrate from another platform" → Lovable oder Bolt.new
4. Bei GitHub auf "Connect" klicken und dieses Repo waehlen.
5. Supabase-Felder leer lassen, falls keine Supabase-DB vorhanden
   (diese App nutzt einen eigenen Node/Express-Server).
6. Import starten.

Variante B (lokal 1:1 starten, ohne Base44):
1. ZIP entpacken
2. cd Projects
3. npm install
4. cd server && npm install && cd ..
5. npm run build
6. ABOWANDLER_STATIC_DIR=dist PORT=8787 node server/index.js
7. Browser: http://127.0.0.1:8787

Demo-Logins stehen in der laufenden App / Seed-Daten
(server/lib/marketSeedFullReset.js), nicht hier dokumentiert.

Struktur:
- src/          Frontend (Vite)
- server/       Backend API (Express)
- electron/     Desktop-Huellen
- dist/         gebautes Frontend
- package.json  Scripts
